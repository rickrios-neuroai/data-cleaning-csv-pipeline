# Data Cleaning & CSV Processing Pipeline

## 📌 Project Overview
This project demonstrates how to clean and preprocess raw CSV data using Python and pandas.

## ⚙️ Features
- Remove duplicate records
- Handle missing values:
  - Mean imputation for numerical data
  - Default values for categorical data
- Data validation using `.info()` and `.describe()`

## 📂 Files
- `data.csv` → Raw dataset
- `cleaned_data.csv` → Cleaned dataset
- `notebook.ipynb` → Full process

## 🛠️ Technologies
- Python
- Pandas
- Jupyter Notebook

## 📊 Output
Clean dataset ready for analysis or machine learning.
